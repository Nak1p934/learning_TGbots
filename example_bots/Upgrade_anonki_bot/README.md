# О боте
Этот бот - фейковые анонимные сообщения, используйте в целях розыграша. Представляет собой обновлённую версию бот + сайт на котором можно наблюдать за жертвами(операторами), + БД.
# Туториал по настройке бота
Выполните `git clone https://github.com/Nak1p934/learning_TGbots`
Перейдите в example_bots/Upgrade_anonki_bot/README.md и пройдите все шаги
# Работа с bot father
Откройте Телеграм и в поиски найдите бота `@BotFather`
Откройти мини апку и создайте своего бота, затем скопируйте токен. ВНИМАНИЕ никому не показывайте токен.
Откройте файл `BOT_TOKEN`, вставьте вместо YOUR_TOKEN свой токен
# Запуск
## Первый запуск
Перейдите в `run.py` и откройте терминал.
Вам нужно активировать и создать виртуальное окружение.
1. Создайте venv - в терминале выполните `python -m venv venv`, если у вас не сработало выполните `python3 -m venv venv`
2. Активируйте venv - в терминале выполните `venv\Scripts\Activate.ps1`, если не сработало - `.\venv\Scripts\Activate.bat`
3. Установите зависимости - в терминале выполните `pip install -r requirements.txt` и дождитесь окончания загрузки
4. Выполните `cd db` и `python create_db.py`
5. Запустите бота - в терминале выполните `python run.py`, если не сработало используйте `python3 run.py`
Вот и всё, бот запущен! Что бы завершить работу в терминале выполните сочитание клавиш ctrl + C.
Теперь запустим сайт
Тут всё в разы проще 
1. Активируйте venv
2. В терминале пропишите `uvicorn main:app --reload`
3. Зажмите Ctrl и кликнете по ip сайта в терминале
## Советы
1. При повторном включении бота не нужно создавать venv и устанавливать зависимости повторно. Просто активируйте виртуальное окружение и запустите бота (Шаги 2 и 4)
2. Если в вашей IDE fastapi подсвечивается как ошибка и не запускается, установите fastapi в общее окружение. Выйдете из venv `deactivate` и выполните `pip install fastapi`. Мне помогло

## Команды
1. start(Без рефералки) - Отпровляет приветствие и просит пользователя отправить /get для того что бы получить ссылку. Так же добавляет его в список пользователей
2. start(С рефералкой) - Запоминает кому пользователь будет отправлять "анонимные сообщения", если пользователь перешёл по своей ссылке ничего работать не будет. Так же дбовляет пользователя в список пользователей
3. clear - отчищает FSM
4. Send_message_all - отпровляет сообщение всем пользователям которые есть в списке users. При перезапуске список отчищается. Использовать: /Send_message_all Your text
5. get - отпровляет ссылку для отправки "анонимных сообщений"
6. Send_ls - отправляет сообщение в лс пользователю. Использовать: /Send_ls user_id text
7. Если отправить просто сообщение то сообщение отправится получателю, но это только если вы переходили по реферальной ссылке. В противном случие бот сообщит об ошибке
8. Если ответить на сообщение в котором содержится информация о отпровителе то ваще сообщение отправится ему
9. /buy Отправить мишку.png
